# Welcome to Bot Traffic generator 👋
![Version](https://img.shields.io/badge/version-0.1.0-blue.svg?cacheSeconds=2592000)

> A basic bot that fetches IP addresses using source and hit a specific URL to create traffic.

## Install

```sh
npm install
```

## Usage

```sh
Make changes to make_hit.js file in order to specify proxy sources/URL to hit
```

## Run tests

```sh
npm run test
```
